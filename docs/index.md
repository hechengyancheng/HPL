## 完整示例速览

### 纯核心语言（不依赖游戏框架）
```h
// 见 core-language-spec.md 附录
set $counter to 0
while $counter is less than 5:
    increase $counter by 1
    echo "Count: " + $counter
```

### 完整游戏（使用所有层次）
```h
// 见 game-framework-spec.md 附录
Room Entrance:
    description is "A dark entrance hall."
    exits is [north to Courtyard]

on game start:
    set player.location to "Entrance"
    echo "Your adventure begins!"

on every turn:
    echo "You are in " + player.location
```
